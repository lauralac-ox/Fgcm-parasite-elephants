# Fgcm-parasite-elephants
Data and R code for Conservation strategies shape stress hormone levels in managed elephants
This repository contains the data, R script and Supplementary Material used in the manuscript:

## Contents

### Data
- `data/dataset_fgcm_modeling.csv`  
  An anonymised, cleaned dataset used for all statistical analyses and figure generation
  presented in the manuscript.

### Code
- `scripts/FGCM-elephant-modeling_Lacomme-etal.R`  
  A single R script that:
  - loads the anonymised dataset,
  - performs all statistical analyses,
  - generates the figures and tables presented in the manuscript and the Supplementary material.

### Supplementary material
- `supplementary_material/Supplementary Material_ Lacomme et al_ FGCMs elephants_vf.docx`  
  Contains all supplementary files associated with the manuscript, including:
  - supplementary tables,
  - supplementary figures.

## Reproducibility

Analyses were conducted in R  version 4.4.2 (2024-10-31 ucrt) -- "Pile of Leaves".  
Required packages are listed at the beginning of each script.

## Data availability and anonymisation

Due to ethical and site-access restrictions, raw data cannot be made publicly available.
The dataset provided here is a cleaned and anonymised version used for all statistical analyses
and figure generation in the manuscript.
